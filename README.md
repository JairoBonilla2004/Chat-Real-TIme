# Chat-Real-TIme
Chat en tiempo real que permita la gesti´on de salas de conversaci´on seguras y colaborativas. El enfoque principal es la creación y gestión de salas de chat por parte de un administrador, con acceso controlado para usuarios mediante PINs, y soporte para dos tipos de salas: texto (solo mensajes) y multimedia (mensajes más subida de archivos).
